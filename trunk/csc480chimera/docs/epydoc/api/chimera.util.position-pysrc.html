 43      There are five factories available, that can be used to create

 45   
 46      Each factory accepts a pair of parameters. The parameters format 
 47      is tied to the choosen frame. The intent is to accept the most 
 48      common form of coordinate pair in each frame. For example, ra/dec 
 49      coordinates are mostly given in 'hms dms' format, so this is the 
 50      default used when strings are passed, however, if integers/floats 
 51      are given, these will be interpreted as 'dd dd' (which is also 
 52      common). 
 53   
 54      However, the common sense isn't common to everyone and to allow 
 55      these different point of views, every factory acceps a Coord 
 56      object as well, which can be created from a bunch of different 
 57      representations. See L{Coord} for more information on the 
 58      available representations. 
 59   
 60      Examples: 
 61

 63      >>> p = Position.fromRaDec('10:00:00', '20:20:20', equinox=Equinox.J2000)

 66      >>> # ok, If you want ra in radians and dec in hours (very strange), use:

 69      No matter which representation is given, the value will be checked 
 70      for range validity. 
 71
