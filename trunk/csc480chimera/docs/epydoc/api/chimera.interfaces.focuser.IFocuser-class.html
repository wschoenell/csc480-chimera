Class IFocuser

source code

Instrument interface for an electromechanical focuser for
astronomical telescopes.

Two kinds of focusers are supported (depending of the driver):

- Encoder based: use optical encoder to move to exact
  positions.
- DC pulse: just send a DC pulse to a motor and move
  to selected directions only (no position information).

moveIn(self, n)

source code 

Move the focuser IN by n steps. Steps could be absolute units (for focuser with absolute encoders) or just a pulse of time. Drivers use internal parameters to define the amount of movement depending of the type of the encoder.

Use moveTo to move to exact positions (If the focuser support it).
Parameters:
  • n (int) - Number of steps to move IN.
Returns: None
Raises:
  • InvalidFocusPositionException - When the request movement couldn't be executed.
