__init__(self)
(Constructor)

source code 

Do object initialization.

Constructor should only do basic initialization. Shouldn't even 'touch' any hardware, open files or sockets. Constructor is called by Manager.
Overrides: object.__init__

Note: Runs on the Manager's thread.

Warning: This method must not block, so be a good boy/girl.

__start__(self)

source code 
Do device initialization. Open files, access drivers and sockets. This method it's called by Manager, just after the constructor.

Note: Runs on the Manager thread.

Warning: This method must not block, so be a good boy/girl.

__stop__(self)

source code 

Cleanup {__start__} actions.

{__stop__} it's called by Manager when Manager is dying or programatically at any time (to remove an Instrument during system lifetime).

Note: Runs on the Manager thread.

Warning: This method must not block, so be a good boy/girl.
