moveIn(self, n)

source code 

Move the focuser IN by n steps.

Driver should interpret n as whatever it support (time pulse or absolute encoder positions). if only time pulse is available, driver must use pulse_in_multiplier as a multiple to determine how much time the focuser will move IN. pulse_in_multiplier and n will be in miliseconds.
Parameters:
  • n (int) - Number of steps to move IN.
Returns: None
Overrides: interfaces.focuserdriver.IFocuserDriver.moveIn

Note: Drivers must raise InvalidFocusPositionException if the request position couldn't be achived.

moveOut(self, n)

source code 

Move the focuser OUT by n steps.

Driver should interpret n as whatever it support (time pulse or absolute encoder positions). if only time pulse is available, driver must use pulse_out_multiplier as a multiple to determine how much time the focuser will move OUT. pulse_out_multiplier and n will be in miliseconds.
Parameters:
  • n (int) - Number of steps to move OUT.
Returns: None
Overrides: interfaces.focuserdriver.IFocuserDriver.moveOut

Note: Drivers must raise InvalidFocusPositionException if the request position couldn't be achived.
