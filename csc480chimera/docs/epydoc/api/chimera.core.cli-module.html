action(*args, **kwargs)

source code 

Defines a command line action with short name 'short', long name 'long'. If 'short' not given, will use the first letter of the method name (if possible), if 'long' not given, use full method name.

Use 'type' if the action require a direct parameter, like '--to 10', in this case, action should be like this:

# @action(long='to', type='int') # def move_to (self, options): # inst.moveTo(options.to)

See Action for information about valid keywork arguments.

parameter(*args, **kwargs)

source code 

Defines a command line parameter with short name 'short', long name 'long'. If 'short' not given, will use the first letter of the method name (if possible), if 'long' not given, use full method name. If type given, parameter will be checked to match 'type'. The default value, if any, shoud be passed on 'default'.

See Parameter for information about valid keywork arguments.
